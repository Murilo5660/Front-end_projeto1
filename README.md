# Site sobre a Revolução Industrial

Este projeto é um site simples desenvolvido para explicar o que foi a Revolução Industrial, abordando seu contexto histórico, principais acontecimentos e impactos na sociedade.

## Funcionalidades

* Página explicativa sobre o que foi a Revolução Industrial
* Apresentação dos principais acontecimentos históricos
* Layout simples e organizado para facilitar a leitura
* Estrutura básica de um site utilizando HTML e CSS

## Pré-requisitos

Para visualizar o projeto, você precisa apenas de:

* Um navegador de internet (Google Chrome, Firefox, Edge, etc.)

## Como executar o projeto

1. Baixe ou clone este repositório:

   ```
   git clone https://github.com/Murilo5660/Front-End_Projeto1
   ```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` em seu navegador.

## Tecnologias

* **HTML5** – Estrutura da página
* **CSS3** – Estilização e layout do site
